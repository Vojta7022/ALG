# ALG – CTU Algorithms

This repository contains my coursework, homework solutions, and exam preparation materials for the CTU Algorithms course.

## About

I keep this repository public mainly as a personal archive of my work and as a study resource for other students.  
It includes homework folders, exam preparation materials, and problem solutions created during the course.

## Contents

This repository may include:
- homework assignments
- algorithmic problem solutions
- exam preparation files
- implementation experiments
- supporting code for course exercises

## Main Topics

The materials in this repository relate to topics such as:
- data structures
- graph algorithms
- dynamic programming
- searching and sorting
- algorithm design and analysis
- problem solving in C++

## Structure

The repository is organized mainly into homework folders such as:
- `HW1` to `HW8`
- `EXAM`

These folders contain my solutions, preparation materials, and related work from different parts of the course.

## Notes

This is primarily a coursework and study repository, not a polished standalone software project.
